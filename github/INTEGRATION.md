# GitHub-only integration

## JavaScript
```bash
npm install github:tellmesh/urirun
```

## Python
```bash
pip install "git+https://github.com/tellmesh/urirun.git@v0.3.5#subdirectory=adapters/python"
```

## Firmware / C
Copy files from `adapters/c/` into the firmware project.

## Recommended repo URL
```txt
https://github.com/tellmesh/urirun
```
